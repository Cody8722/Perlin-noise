# RPG 世界生成器

使用 Perlin Noise 演算法生成程序化 RPG 遊戲地形的互動式網頁工具。

## 功能特色

- 🌍 **程序化地形生成** - 使用 Perlin Noise 演算法生成自然隨機的地形
- 🎨 **多樣生物群系** - 包含海洋、沙灘、草原、森林、沙漠、雪地等多種地形
- ☁️ **動態雲層** - 實時動畫雲層效果
- 🎛️ **即時參數調整** - 可調整縮放、細節、海平面、降雨量等參數
- 📱 **響應式設計** - 支援桌面和行動裝置
- 🖱️ **互動式地圖** - 滑鼠懸停顯示生物群系資訊

## 使用方式

1. **開啟網頁**

   直接用瀏覽器開啟 `index.html` 即可

   ```bash
   # 使用 Python 快速啟動本地伺服器（推薦）
   python -m http.server 8000

   # 然後在瀏覽器訪問
   # http://localhost:8000
   ```

2. **控制面板操作**

   - **生成新世界** - 點擊按鈕產生全新的隨機地形
   - **縮放 (Scale)** - 調整地形的整體尺度（10-150）
   - **細節 (Octaves)** - 控制地形細節層數（1-6）
   - **海平面** - 調整海洋與陸地的比例（0.1-0.8）
   - **降雨量** - 影響生物群系的濕度分布（-0.4 到 0.4）
   - **顯示雲層** - 切換動態雲層顯示

3. **互動功能**

   - 將滑鼠懸停在地圖上可查看該位置的生物群系類型

| ❄️ 雪地 | 高海拔 + 高濕度 |

## 演算法原理

### Perlin Noise

Perlin Noise 是由 Ken Perlin 在 1983 年開發的梯度噪聲演算法，廣泛應用於程序化內容生成。

**核心特性：**
- 連續性：相鄰值變化平滑
- 可重複性：相同種子產生相同結果
- 自然感：類似真實世界的隨機分布

### 分形布朗運動 (FBM)

透過疊加多個不同頻率和振幅的 Perlin Noise，產生更複雜自然的地形：

```javascript
FBM(x, y) = Σ(噪聲(x×頻率ᵢ, y×頻率ᵢ) × 振幅ᵢ)
```

每一層（八度）：
- 頻率加倍（細節更精細）
- 振幅減半（影響力降低）
